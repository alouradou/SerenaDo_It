Bienvenue sur SérénaDo_It !

Options de personnalisation

1. Choisir le fichier Excel avec ses matières

Une fois le fichier Excel uploadé, SérénaDo_It va automatiquement créer un lien d'abonnement personnalisé pour votre calendrier. Il vous suffira de copier ce lien et de l'ajouter à votre application favorite.

2. Choisir un fichier dans le github de Do_It

Méthode stable.

On va se sevir ici du travail de Duc pour obtenir directement les fichiers excels générés.

Choisir son fichier dans la liste

3. Choisir son nom dans la liste déroulante !

La liste des élèves est acquise depuis la Google Spreadsheet du choix des matières.

Ensuite, on télécharge les matières ou chaque élève n'a pas renseigné de croix (x/X).

Enfin, on supprime les cours dont le nom correspond à ces matières avec le script de Duc.
